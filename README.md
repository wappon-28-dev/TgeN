# TgeN

Creation of templates for various projects / 色々なプロジェクト用のテンプレート作成

## Description / 説明

プロジェクトのフォルダーの雛形を対話式で作成できます  
JSON ファイルに雛形を先に構成することができます  
フォルダーのアイコンも設定できます
